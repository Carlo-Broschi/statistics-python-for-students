**問3. クラスのダミーを足すと自由度調整済みR²は？**
```python
smf.ols('英語 ~ 数学 + 国語 + 勉強時間 + C(クラス)', data=df).fit().rsquared_adj
```
💡 効かない変数を足すと、自由度調整済みR²は下がることがある（過剰な変数のペナルティ）。


**問4. 変数追加とR²／自由度調整済みR²**
```python
import statsmodels.api as sm
for cols in [['勉強時間'],['勉強時間','国語']]:
    r=sm.OLS(df['数学'],sm.add_constant(df[cols])).fit()
    print(cols,'R²',round(r.rsquared,3),'adjR²',round(r.rsquared_adj,3))
```
💡 素のR²は変数を足すと必ず増える。自由度調整済みR²は役に立たない変数では下がるので、モデル比較は後者で行う。
