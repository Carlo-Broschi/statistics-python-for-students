**問3. パレート図**
```python
import pandas as pd
s = pd.Series({'部品A':60,'部品B':25,'部品C':10,'その他':5}).sort_values(ascending=False)
cum = s.cumsum()/s.sum()*100
print(cum)   # 上位2項目(部品A,B)で85% → 8割超
```
💡 「上位の少数で全体の大半を占める」＝パレートの法則。対策の優先順位づけに使う。
