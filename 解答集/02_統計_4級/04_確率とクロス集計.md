**問1. トランプの確率**
```python
print(13/52, 12/52)   # ハート=1/4=0.25, 絵札=12/52≈0.231
```
💡 確率＝(あてはまる枚数)÷(全52枚)。

**問3. サイコロの期待値**
```python
import numpy as np
v = np.arange(1,7); p = np.full(6, 1/6)
print(np.sum(v*p))   # 3.5
```
💡 期待値は「1〜6を平均したら3.5」。実際に3.5は出ないが、何回も振った平均が3.5に近づく。


**問4. 非復元抽出のシミュレーション**
```python
import numpy as np
rng=np.random.default_rng(0); box=['当','当','×','×','×']
hit=np.mean([list(rng.choice(box,2,replace=False)).count('当')==2 for _ in range(10000)])
print('シミュ',round(hit,3),' 理論',round(2/5*1/4,3))
```
💡 理論値は 2/5×1/4=0.1。非復元では1本目で当たりが1本減るぶん、2本目の当たる確率が下がる。
