# 05_表示回数とクリック数の関係（単回帰） ― 解答例（解説つき）

```python
import statsmodels.api as sm
# 課題1: クリック数 → 獲得数 の単回帰
m2 = sm.OLS(mk['獲得数'], sm.add_constant(mk['クリック数'])).fit()
print('相関 r =', round(mk['クリック数'].corr(mk['獲得数']), 3), ' R² =', round(m2.rsquared, 3))
# → 表示→クリック（r≈0.89）と違い、クリック→獲得は相関が弱い/ほぼ無い。
#    高クリックのWeb広告ほど獲得率(CVR)が低いなど、チャネルで効率が違うため
#    （3級02のシンプソンのパラドックスと同じ構図）。

# 課題2: チャネルで色分けした散布図
fig, ax = plt.subplots(figsize=(7,4.5))
for ch, d in mk.groupby('チャネル'):
    ax.scatter(d['表示回数'], d['クリック数'], label=ch)
ax.set_xlabel('表示回数'); ax.set_ylabel('クリック数'); ax.legend(); plt.show()
# → チャネルごとに傾き(CTR)が違う。全体の1本の直線は「平均的なCTR」にすぎない。
```

**課題3（考察）**：R² が同じでも、散布図に **曲線的な関係**や**外れ値**があると直線当てはめは不適切。例：頭打ち（逓減）する関係を直線で当てると、両端で大きく外す。必ず散布図と残差を確認する。
