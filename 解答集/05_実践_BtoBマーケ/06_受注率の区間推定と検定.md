# 06_受注率の区間推定と検定 ― 解答例（解説つき）

```python
import numpy as np
from scipy import stats

# 課題1: 「紹介」チャネルだけの受注率と95%信頼区間
sub = btob[btob['獲得チャネル']=='紹介']
n = len(sub); p = sub['受注'].mean(); se = np.sqrt(p*(1-p)/n)
print(f'紹介: 受注率 {p:.1%}  95%CI [{p-1.96*se:.1%}, {p+1.96*se:.1%}]  (n={n})')
# → 全体より高め。ただし n が小さいぶん区間は広い（推定が甘い）。

# 課題2: 目標を25%にして検定
p0 = 0.25; n_all = len(btob); p_all = btob['受注'].mean()
z = (p_all - p0)/np.sqrt(p0*(1-p0)/n_all)
print('z =', round(z,3), ' p値 =', round(2*(1-stats.norm.cdf(abs(z))),4))
# → 目標を25%に上げると「超えた」とは言いにくくなる（p値が大きくなる）。

# 課題3: 半幅±3%以内に必要な n（p=0.25 で見積もり）
p = 0.25
print('必要なn ≈', np.ceil((1.96/0.03)**2 * p*(1-p)))
```

💡 半幅 `1.96√(p(1−p)/n) ≤ 0.03` を n について解くと `n ≥ (1.96/0.03)² p(1−p)`。p が不明なら最も n が大きくなる p=0.5 で安全側に見積もる。
