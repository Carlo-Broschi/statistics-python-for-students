**問1. 国語の不偏標準偏差と変動係数**
```python
print(df['国語'].std())                       # 不偏標準偏差（÷(n-1)）
print(df['国語'].std() / df['国語'].mean())    # 変動係数
```
💡 **なぜ変動係数？** 標準偏差は単位つき（点）なので、平均の大きさがちがうデータ同士の
「散らばりの大きさ」は単純に比べられない。平均で割って**相対化**すると比較できる。

**問2. 英語を標準化 → 平均0・標準偏差1**
```python
z = (df['英語'] - df['英語'].mean()) / df['英語'].std()
print(round(z.mean(), 5), round(z.std(), 3))   # ≈ 0, ≈ 1
```
💡 標準化は「平均を引いて標準偏差で割る」操作。**定義そのものから**、変換後は必ず
平均0・標準偏差1になる（だから単位の違うデータを同じ土俵で比べられる）。

**問3. 同じ75点、偏差値が高いのは？**
```python
for c in ['数学', '英語']:
    print(c, round(50 + 10 * (75 - df[c].mean()) / df[c].std(), 1))
```
💡 同じ75点でも、**平均が低く・σが小さい科目**ほど偏差値は高い。
偏差値は「点数の絶対値」ではなく「集団の中での相対位置」を表すから。

**問4.（記述）なぜ標準偏差を使う？**
分散は偏差を2乗するので単位が「元の単位の2乗」（例：点²）になり、現実の量として解釈しづらい。
√をとって標準偏差にすると単位が元（点）に戻り、「平均からおおよそ何点ズレているか」と
**直感的に解釈できる**。だから散らばりの代表的な指標には標準偏差を使う。
