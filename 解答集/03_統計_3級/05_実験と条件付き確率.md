# 05_実験と条件付き確率 ― 解答例（解説つき）

**問1. 交絡と実験**
アプリ利用と成績の両方に効く第3の要因（もともとの生活習慣・意識など）が**交絡**。
だから観察だけでは「アプリが成績を上げた」とは言えない。
→ 無作為に2群に分け、片方だけアプリを使わせて成績を比べる**実験**で因果を検証する。

**問2. 独立な事象**
```python
print((1/2)*(1/6))   # 1/12
```
💡 独立なら確率はかけ算。

**問3. 条件付き確率**
```python
print(60/180)   # 1/3
```
💡 「朝食なし」に絞った中での「運動部」の割合＝クロス表の行比率。
