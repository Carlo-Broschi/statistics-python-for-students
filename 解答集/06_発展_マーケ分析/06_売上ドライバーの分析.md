# 06_売上ドライバーの分析（重回帰） ― 解答例（解説つき）

```python
import statsmodels.formula.api as smf

# 問1: 単回帰 vs 重回帰 の R²・自由度調整済みR²
m1 = smf.ols('売上 ~ 新規リード', data=kpi).fit()
m2 = smf.ols('売上 ~ 新規リード + 受注数', data=kpi).fit()
print(f'単回帰  R²={m1.rsquared:.3f} 調整済み={m1.rsquared_adj:.3f}')
print(f'重回帰  R²={m2.rsquared:.3f} 調整済み={m2.rsquared_adj:.3f}')
# → 受注数を足すとR²は上がる。調整済みR²も上がるなら、足す価値があった変数。

# 問2: 売上を「受注数」だけ / 「新規リード」だけ で単回帰してR²を比較
ma = smf.ols('売上 ~ 受注数', data=kpi).fit()
mb = smf.ols('売上 ~ 新規リード', data=kpi).fit()
print('売上~受注数   R² =', round(ma.rsquared, 3))
print('売上~新規リード R² =', round(mb.rsquared, 3))
# → R²が高いほうが、その1要因だけで売上をよく説明できている。
```

**問3（考察）**：新規リードと受注数は強く相関する（多重共線性）。このとき各偏回帰係数は不安定で、符号や大きさが入れ替わることもある。個々の係数を断定的に読まず、`売上 ~ 新規リード` のように1つずつ入れた結果とも見比べる。VIF で共線性を数値確認するのも有効。
