# 02_売上トレンドの分析 ― 解答例

```python
from statsmodels.graphics.tsaplots import plot_acf
# 問1: 受注数で同じ分析
kpi['受注_前年比%']=kpi['受注数'].pct_change(12)*100
kpi['受注数'].rolling(12).mean().plot(); plt.show()
plot_acf(kpi['受注数'], lags=24); plt.show()

# 問2: 3か月 vs 12か月 移動平均
kpi['MA3']=kpi['売上'].rolling(3).mean()
kpi[['売上','MA3','移動平均12']].plot(figsize=(10,4)); plt.show()
# MA3は短期の動きに敏感、MA12は大きなトレンドだけを残す。
```

問3（考察）：前月比が大きく上下するのに前年比が安定しているのは、毎年同じ月に増減する「季節変動」が主因のとき。前年同月と比べる前年比は季節を相殺するため安定する。
