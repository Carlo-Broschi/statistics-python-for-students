# 05_チャネル別獲得率の比較（分散分析） ― 解答例（解説つき）

```python
from scipy import stats
# 問1: CTR でチャネル間ANOVA
mk['CTR'] = mk['クリック数']/mk['表示回数']
print('CTRのANOVA:', stats.f_oneway(*[g['CTR'].values for _, g in mk.groupby('チャネル')]))

# 問2: CVR最高と最低のチャネルで2標本t検定 → F=t²
mk['CVR'] = mk['獲得数']/mk['クリック数']
hi = mk[mk['チャネル']=='紹介']['CVR']; lo = mk[mk['チャネル']=='Web広告']['CVR']
t = stats.ttest_ind(hi, lo, equal_var=True).statistic       # 等分散仮定のt
F = stats.f_oneway(hi, lo).statistic
print('t² =', round(t**2, 3), ' F =', round(F, 3), ' → 一致')
```

**問3（考察）**：ANOVAは「どこかに差がある」ことしか言わない。**どのチャネル間に差があるか**を知るには、テューキーのHSDなどの**多重比較法**（`statsmodels` の `pairwise_tukeyhsd`）を使う。
