# 04_チャネル別クリック率の比較（2標本t検定） ― 解答例（解説つき）

```python
from scipy import stats
mk['CTR'] = mk['クリック数']/mk['表示回数']

# 問1: 展示会 vs メルマガ のCTRをt検定
a = mk[mk['チャネル']=='展示会']['CTR']; b = mk[mk['チャネル']=='メルマガ']['CTR']
print('展示会 vs メルマガ:', stats.ttest_ind(a, b, equal_var=False))

# 問2: CVR（獲得数/クリック数）で 紹介 vs Web広告
mk['CVR'] = mk['獲得数']/mk['クリック数']
a = mk[mk['チャネル']=='紹介']['CVR']; b = mk[mk['チャネル']=='Web広告']['CVR']
print('CVRで比較:', stats.ttest_ind(a, b, equal_var=False))
```

**問3（考察）**：p値が 0.01〜0.05 の間にあるとき、有意水準5%なら「有意」だが1%なら「有意でない」と結論が変わる。基準を厳しくするほど偽陽性は減るが、本当の差を見逃すリスク（第2種の過誤）は増える。
