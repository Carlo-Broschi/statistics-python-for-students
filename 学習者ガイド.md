# 学習者ガイド（はじめに）

Python で統計検定（4〜2級）の概念を、手を動かして学ぶ教材です。番号順に進めるのがおすすめです。

## 1. ノートの動かし方（どちらか）
- **Google Colab（かんたん・おすすめ）**：各ノート上部の「Open in Colab」バッジをクリックして開く。インストール不要。
- **ローカル**：`uv sync` → `uv run jupyter lab`。

## 2. 各ノートの進め方
1. 先頭の「① セットアップ」セルを**最初に実行**する（データを自動で準備）。実行し忘れると後のセルでエラーになる。
2. 上から順に、説明 → コード → 確認テスト。確認テストは `ans` を自分の答えで置き換えて実行すると、その場で正誤が出る。
3. 末尾の「練習問題」を解く。解答例は `解答集/` にリンクがある。

## 3. 学習者コードの設定（最初に一度だけ）
講座から配られた**自分のコード**を一度だけ設定すると、各ノートの完了が記録されます。

- **Colab**：左の鍵アイコン（シークレット）→ 新規追加 → 名前を `STUDENT_CODE`、値に自分のコードを入れ、「ノートブックからのアクセス」をオンにする。一度だけで、自分のすべてのColabノートに有効。
- **ローカル**：Jupyter 起動前に `export STUDENT_CODE='自分のコード'` を設定する。または、初回に末尾の完了ログセルを実行すると入力欄が出るので、コードを貼ると保存され次回から不要。

各ノートを終えたら、**末尾の「完了ログ」セルを実行**すると、コードが自動送信されて記録されます（名前の再入力は不要）。

> コードは自分専用です。他人に教えないでください（教えると別人になりすませてしまいます）。

## 4. グラフの日本語が文字化けするとき
各ノート先頭セルにあるフォント指定（Mac / Windows / Linux）の該当行を有効化する。
