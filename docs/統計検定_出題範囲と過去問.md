# 統計検定 出題範囲・過去問リファレンス（4級／3級／2級）

本教材が準拠する**統計検定**（日本統計学会公式認定）の出題範囲・試験形式・過去問入手先。範囲表は公式PDF（2018/12/14版、2020年4月実施〜）から抽出。

> **著作権について**：過去問題・例題の**本文は著作物**であり、ここには転載しません。
> 本書では公式が公開している**出題範囲（用語リスト）**のみを整理し、実際の過去問は下記の正規入手先を案内します。
> 本教材の練習問題は、出題範囲に沿って独自に作成した「過去問“風”」のオリジナル問題です。

---

## 0. 各級の試験概要（CBT方式・2026年6月時点）

| | 4級 | 3級 | 2級 |
|---|---|---|---|
| 想定レベル | 中学レベルのデータ活用 | 高校〜大学初級／データ分析 | 大学基礎課程（1・2年） |
| 試験方式 | CBT | CBT | CBT |
| 試験時間 | 60分 | 60分 | 90分 |
| 問題数 | 30問程度 | 30問程度 | 35問程度 |
| 形式 | 4〜5肢選択 | 4〜5肢選択 | 4〜5肢選択 |
| 合格基準 | 100点中 **60点** | 100点中 **65点** | 100点中 **60点** |
| 受験料（一般） | 5,000円 | 6,000円 | 7,000円 |
| 受験料（学割） | 3,500円 | 4,000円 | 5,000円 |

出題は3つの観点で構成：**統計リテラシー**（用語・定義）／**統計的推論**（用語間の関連・解釈）／**統計的思考**（文脈の中での活用）。

## 1. 統計検定4級 出題範囲表

| データの集計 | 度数分布表 | 度数分布表、度数、相対度数、階級、階級値、階級幅、累積度数、累積相対度数、度数分布表からの統計量の求め方 |

## 3. 統計検定2級 出題範囲表

対象：大学基礎課程。記述統計から推測統計（推定・検定・回帰・分散分析）まで。

| 大項目 | 小項目 | 学習しておくべき用語 |

| 1変数データ | データの分布の記述 | 質的変数（カテゴリカル）、量的変数（離散型・連続型）、棒グラフ、円グラフ、幹葉図、度数分布表・ヒストグラム、累積度数グラフ、分布の形状（右/左に裾、対称、ベル型、一様、単峰、多峰） |
| 1変数データ | 中心傾向の指標 | 平均値、中央値、最頻値（モード） |
| 1変数データ | 散らばりなどの指標 | 分散（n−1で割る）、標準偏差、範囲、四分位範囲、箱ひげ図、ローレンツ曲線、ジニ係数、カイ二乗値、歪度、尖度 |
| 1変数データ | 中心と散らばりの活用 | 偏差、標準化（z得点）、変動係数、指数化 |
| 2変数以上のデータ | 散布図と相関 | 散布図、相関係数、共分散、層別散布図、相関行列、みかけの相関（擬相関）、偏相関係数 |
| 2変数以上のデータ | カテゴリカルデータ | 度数表、2元クロス表 |
| 2変数以上のデータ | 単回帰と予測 | 最小二乗法、変動の分解、決定係数、回帰係数、分散分析表、観測値と予測値、残差プロット、標準誤差、変数変換 |
| データの活用 | 時系列データの処理 | 成長率、指数化、幾何平均、系列相関・コレログラム、トレンド、平滑化（移動平均） |
| 推測のためのデータ収集法 | 観察研究と実験研究 | 観察研究、実験研究、調査の設計、母集団、標本、全数調査、標本調査、無作為抽出 |
| 推測のためのデータ収集法 | 標本調査と無作為抽出 | 標本サイズ、標本誤差、偏りの源、標本抽出法（系統・層化・クラスター・多段抽出法） |
| 推測のためのデータ収集法 | 実験 | 実験のデザイン（実験計画）、フィッシャーの3原則 |
| 確率モデルの導入 | 確率 | 事象と確率、加法定理、条件付き確率、乗法定理、ベイズの定理 |
| 確率モデルの導入 | 確率変数 | 離散型・連続型確率変数、期待値・分散・標準偏差、和と差（同時分布、和の期待値・分散）、共分散・相関 |
| 確率モデルの導入 | 確率分布 | ベルヌーイ試行、二項分布、ポアソン分布、幾何分布、一様分布、指数分布、正規分布、2変量正規分布、超幾何分布、負の二項分布 |
| 推測 | 標本分布（標本統計量） | 独立試行、標本平均の期待値・分散、チェビシェフの不等式、大数の法則、中心極限定理、二項分布の正規近似、連続修正、母集団、母数 |
| 推測 | 標本分布（正規母集団） | 標準正規分布・分布表の利用、t分布、カイ二乗分布、F分布、上側確率点（パーセント点） |
| 推測 | 推定 | 点推定、推定量と推定値、一致性、不偏性、信頼区間、信頼係数、母平均・母分散・母比率・相関係数の区間推定、2標本の区間推定 |
| 推測 | 仮説検定 | 仮説検定の理論、p値、帰無仮説H₀と対立仮説H₁、両側・片側検定、第1種・第2種の過誤、検出力、母平均・母分散・母比率の検定、2標本の検定（分散既知/未知・等分散ほか）、適合度検定、独立性の検定 |
| 線形モデル | 回帰分析 | 回帰直線の傾きの推定と検定、重回帰モデル、偏回帰係数、回帰係数の検定、多重共線性、ダミー変数回帰、自由度調整済み決定係数 |
| 線形モデル | 実験計画の理解 | 実験、処理群と対照群、反復、ブロック化、一元配置実験、3群以上の平均の差（分散分析）、F比 |
| 活用 | 統計ソフトウェアの活用 | 計算出力の活用、問題解決への活用 |

**→ 本教材の対応**：`04_統計_2級/`（01 区間推定／02 仮説検定／03 カイ二乗と分散分析／**04 母分散・F検定・母比率の差**／**05 いろいろな確率分布とベイズ**／**06 重回帰分析**／**07 抽出法・実験計画・格差指標**）。
実データ演習は `05_実践_BtoBマーケ/`（回帰・クロス集計・カイ二乗・A/Bテスト）。

追加でカバー済み：母分散の検定・分散比(F)・母比率の差の検定・等分散/Welchの使い分け（04）／ポアソン・幾何・一様・指数分布・ベイズの定理（05）／重回帰・偏回帰係数・決定係数・回帰係数の検定・ダミー変数・多重共線性VIF（06）／各種標本抽出法・フィッシャーの3原則・歪度尖度・ローレンツ曲線とジニ係数（07）。

> **なお残る発展項目（公式テキスト併用推奨）**：2変量正規分布・超幾何分布・負の二項分布、相関係数の区間推定、時系列のコレログラム/幾何平均、チェビシェフの不等式など細目。主要範囲は本教材でカバー済み。
