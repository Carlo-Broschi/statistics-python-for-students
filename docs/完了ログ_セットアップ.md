# 完了ログのセットアップ（講師用）

各ノート末尾の「完了ログ」セルは、学習者が実行すると **名前・ノート名・日時** を Google スプレッドシートに1行追記する。学習者は Google 認証不要（下記の Web App URL に POST するだけ）。

## 仕組み
学習者のノート → `LOG_URL`（Apps Script Web App）へ POST → スプレッドシートに追記。

## 手順

### 1. スプレッドシートと Apps Script を用意
1. 新しい Google スプレッドシートを作る。
2. 拡張機能 → Apps Script を開き、以下を貼り付けて保存。

```javascript
function doPost(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName('log') || ss.insertSheet('log');
  const d = JSON.parse(e.postData.contents);
  sheet.appendRow([d.time, d.name, d.notebook]);
  return ContentService.createTextOutput('ok');
}
```

### 2. Web App としてデプロイ
- デプロイ → 新しいデプロイ → 種類「ウェブアプリ」。
- 「次のユーザーとして実行」= **自分**、「アクセスできるユーザー」= **全員**。
- 発行された **ウェブアプリ URL**（`https://script.google.com/macros/s/XXXX/exec`）をコピー。これが `LOG_URL`。

### 3. 全ノートに URL を設定して配布
リポジトリのルートで次を実行すると、全ノートの `LOG_URL` を一括設定できる。

```python
import nbformat, glob
url = "https://script.google.com/macros/s/XXXX/exec"   # ← 2でコピーしたURL
for f in glob.glob('**/*.ipynb', recursive=True):
    if '/.' in f: continue
    nb = nbformat.read(f, as_version=4)
    for c in nb.cells:
        if c.cell_type == 'code' and 'LOG_URL = ""' in c.source:
            c.source = c.source.replace('LOG_URL = ""', f'LOG_URL = "{url}"')
    nbformat.write(nb, f)
```

設定後にコミット・配布すれば、学習者は `NAME` を入れて最後のセルを実行するだけで記録される。

## 集計
- スプレッドシートの `log` シートに `日時 / 名前 / ノート名` が溜まる。
- ピボットテーブルで「名前 × ノート名」を作れば、誰がどこまで進めたかを一覧できる。

## 注意
- Web App は未認証 POST を受けるため、URL を知る第三者が書き込める。少人数の講座なら実用上問題ないが、厳密に守るなら学習者ごとのトークンや Google Forms 方式に切り替える。
- 記録されるのは学習者が入力した名前・ノート名・日時のみ。個人情報の扱いは配布時に周知する。
